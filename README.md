## 项目简介

- 本项目旨在为简中自由软件本地化工作者提供一般性指引
- 本项目由发行自[安同开源社区](https://github.com/AOSC-Dev/)的[《大陆简中自由软件本地化工作指南》（PDF 版本）](https://repo.aosc.io/aosc-l10n/zh_CN_l10n.pdf)演变而来

## 项目站点职能（筹）

- 提供大陆简中自由软件本地化工作指南文档
- 提供简中本地化工作相关的博客内容分享
